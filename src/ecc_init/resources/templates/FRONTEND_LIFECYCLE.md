# Frontend GSD Lifecycle

Use these commands when a project has the frontend Pack enabled and GSD is initialized:

```text
/gsd-discuss-phase N
/gsd-ui-phase N
/gsd-plan-phase N
/gsd-execute-phase N
/gsd-verify-work N
/gsd-ui-review N
```

Keep domain-specific design, implementation, and verification guidance in the installed frontend skills. Keep phase orchestration in GSD.
