# Vercel Platform

Use this skill when a frontend project targets Vercel or uses Vercel previews.

Focus on:

- Build command, output directory, framework preset, and environment variable checks.
- Preview deployment assumptions and production-readiness review.
- Edge/runtime caveats for Next.js, React, server components, and API routes.
- Safe handling of secrets, preview URLs, and deployment-specific configuration.
- Clear fallback guidance when a project is not configured for Vercel.

This skill is advisory and project-facing. It does not run deployments or install the Vercel CLI by itself.
