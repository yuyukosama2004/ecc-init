# Playwright Quality

Use this skill when frontend work needs browser verification, screenshots, or regression checks.

Focus on:

- Choosing stable user flows and assertions for browser tests.
- Capturing desktop and mobile screenshots for visual review.
- Checking canvas, media, hover, focus, loading, and responsive states.
- Keeping tests deterministic by controlling network, time, and test data.
- Reporting failures with the smallest reproducible viewport and interaction path.

This skill describes verification expectations. It does not install Playwright or create a test runner automatically.
