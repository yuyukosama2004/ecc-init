# UI UX Pro Max

Use this skill for product-facing frontend work where layout, interaction quality, accessibility, and visual polish matter.

Focus on:

- Information architecture that matches the user's actual workflow.
- Responsive layouts with stable dimensions and no overlapping text.
- Accessible controls, visible focus states, labels, and semantic structure.
- Visual QA criteria that can be checked with screenshots or browser automation.
- Practical design critique for spacing, hierarchy, density, and empty/loading/error states.

Do not define project workflow phases here. Keep orchestration in the active workflow system and keep this skill focused on frontend product quality.
