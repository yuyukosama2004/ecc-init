## Deferred Boundaries

- `ecc-init apply --yes` writes bundled and explicit pinned GitHub archive project-scope files transactionally; global components and real external installers remain guarded or unsupported.
- `ecc-init update --sources` verifies declarations and previews source updates; it does not fetch and project remote source content during ordinary local runs.
- `ecc-init remove` edits managed GSD config bindings only; it does not delete user files or uninstall GSD Core.
